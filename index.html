Fundamental HTML Tags

What are HTML Tags and why are they important?

When diving into the world of web development, HTML is often the first language you will come across. Therefore, it is vital to know the most basic HTML tags and understand their meaning. Take a look at the table below to learn more about HTML Tags!

Fundamental HTML Tags Table
Tag Tag Name Meaning
<head>...</head> The document head The head element contains key information about the web page
<title>...</title> The page title The title element contains the title of the page
<body>...</body> The page's content Comes after the head element and contains the web page content
<h1>...</h1> Headings Headings help you break up your page. There are six HTML headings, h1 is the biggest, while h6 is the smallest
<p>...</p> Paragraph The p element allows you to create paragraphs of text
<a>...</a> Link The a element lets you create links to other content - whether that be on your page or another site
<img> Image The img element is a self closing tag which allows you to upload images to your site
<div>...</div> Block-level container The div element is a container which helps to break up content on your page
<span>...</span> Inline container The span element is similar to the div tag, however it is designed to hold smaller pieces of content
